    Mutations

    Abstract class

    Abstract class responsible for applying mutations to cells.

    duplicate_and_mutate_attribute(self, cell, attr_name)

    Duplicates and mutates the attribute attr_name.

    The attribute is duplicated so that its first dimension matches cell.n_cells. After that, it is mutated to induce variations between cells.

    Parameters:

    Name Type Description Default
    cell Cell

    Cell.
