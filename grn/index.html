    Gene Regulatory Networks

    GRN

    Abstract class representing the Gene Regulatory Network (GRN) of the Cell.

    It inherits from the networkx.DiGraph class. Nodes typically represent genes, and edges typically represent a regulation relationship between a regulator gene (or transcription factor, TF) and a regulated gene.

    Random GRN

    Class implementing a Gene Regulatory Network (GRN) initialized at random. It inherits from the GRN class.

    The elements of the adjacency matrix are sampled independently at random in {0, 1}, given the number of nodes, and the average number of parents per node. The resulting graph can contain cycles and self loops. In addition, the initializer makes sure that each node has at least one parent.
